how to establish github comments in blogs

There are two main ways to implement a blog comment system using the github api.

  1. in Replacing Disqus with Github Comments, first set a related issue, and then in the blog page you can see the relevant comments, and then jump to the corresponding github issues page to comment.

  2. in Gitment Do not leaving the blog pages to comments by OAuth.

I have to open a new page to writte a comments in first method. But there are security problems in second method.
